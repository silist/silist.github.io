佳能LBP2900打印机不断卡纸修理

发表于 2020-02-23 分类于 折腾

起因

家里超期服役的佳能LBP2900打印机终于不堪重负的坏掉了。

现象

主要症状为：一次性打印超过一页的文件，在第二张纸打印时必然会停止然后报卡纸。
观察发现打印机在取纸过程中不停取纸，直至卡纸。区别于正常打印时打印完一张纸才去取第二张纸。

解决方案

查找方案

由于打印过程十分复杂，涉及众多传感器联合工作，任何一个部分出问题都可能导致卡纸，一时间没有思路。
碰运气去搜索”LBP2900 不停取纸”时发现了这篇文章：
(佳能LBP2900打印机连续进纸和卡纸完美解决方法)[https://jingyan.baidu.com/article/546ae1852833841148f28c43.html]
症状和我家的打印机完全一致。文中解释为继电器降噪海绵退化导致粘连。

动手拆机

这篇文章的拆机过程比较简单，实际参考了;
(LBP2900拆机过程)[https://wenku.baidu.com/view/b3f8a9257375a417866f8fd3.html]
过程大概是:

  1. 打开侧板卡扣，用平口螺丝刀翘一圈取下侧板。
  2. 取下背板螺丝并取下背板。过程中需要大力出奇迹。
  3. 取下继电器螺丝并取出继电器。
  4. 取下继电器弹簧，清理海绵部分。我先用湿巾擦拭后用橡皮仔细擦掉剩余部分。
  5. 复原打印机。

    修理结果

    经过以上死马当作活马医的步骤后，打印机重新工作后不再卡纸。看来这一问题可能是该型号打印机的通病。
    另外硒鼓也出了点问题导致打印中间部分区域不清晰，回头考虑换个硒鼓。

按之字形打印二叉树

发表于 2020-02-22 分类于 题目

剑指offer中的题。

题目描述

请实现一个函数按照之字形打印二叉树，即第一行按照从左到右的顺序打印，第二层按照从右至左的顺序打印，第三行按照从左到右的顺序打印，其他行以此类推。

解法思路

使用久而不用的C++写了这一题。忘了很多东西，实现时参考了别人的代码…
注意stack的push是从栈顶压入新元素，pop是从栈顶弹出新元素。LIFO。
一层一层处理。
设置两个栈，root节点放入奇数栈。
奇数栈子节点弹出到偶数栈中，加新节点先加左子节点再加右子节点；
偶数栈子节点弹出到奇数栈中，加新节点先加右子节点再加左子节点。
