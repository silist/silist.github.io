在阿里云ECS上搭建blog

发表于 2020-02-22 更新于 2020-02-23 分类于 折腾

动机

由于最近疫情大家都放假在家，阿里云搞了一个活动送一年ECS，配置2C4G1M。总体来说比之前腾讯云的学生机配置高了一档。申请之后为了让其发挥价值，决定搭一个blog。

选型

首先到v2ex上搜了一下，发现推荐比较多的是Hexo。相比较Wordpress，Hexo更轻量，需要资源更少，适合低配ECS环境。

环境

ECS： Ubuntu 18.04 x64

NeXT本身提供了四种主题，比较之后选择了双栏主题Pisces。
